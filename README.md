# HHManager
Scrumprosjekt 2018
## Oppstart
Instruksjonene vil gi deg en kopi av prosjektet på din pc for utvikling og testformål. Se under for hvordan installasjon og test foregår.
Tingene du trenger å installere:
...
1. Java 8 JDK, utviklerverktøyet som er brukt i HHManager.
2. Apache Maven, for å administrere prosjektets bygging, rapportering og dokumentasjon.
3. En database server, for å lagre data fra systemet. Det krever et datakoblings bibliotek for å etablere kobling mellom din database og systemet.
4. Git, en distribuert versjonskontroll, som nødvending for å klone prosjektet til din egen datamaskin for å kjøre programmet. 
...
